# 📄 Document Scanner & OCR using Computer Vision

## 📌 Problem Statement

In everyday life, students and professionals often need to digitize handwritten or printed documents. Traditional scanners are not always available, and manual typing is time-consuming. This project solves this problem by automatically scanning documents from images and extracting text.

---

## 🚀 Overview

This project uses Computer Vision techniques to detect a document in an image, convert it into a scanned format, and extract text using OCR (Optical Character Recognition).

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Tesseract OCR

---

## ⚙️ How It Works

1. Upload an image
2. Detect edges using Canny Edge Detection
3. Find document contour
4. Apply perspective transform
5. Convert to black & white
6. Extract text using OCR

---
